# 前端与 GUI 设计高阶规范体系

本技能融合了业界顶级的设计语言逻辑与现代前端工程实践，旨在确保最终产出的 UI 界面不仅具备规范的底层实现，同时展现出高端的视觉美学（Premium Design）与极致的用户体验。

## 1. 技术栈基石与工程约束

- **核心选型**：使用 HTML 构建语义化结构，JavaScript 驱动交互逻辑。除明确要求外，默认使用原生 Vanilla CSS 保持灵活性。若需要构建复杂 Web 应用，建议采用 Next.js 或 Vite。
- **项目初始化机制**：使用框架时务必通过 `npx -y`（配合 `--help` 查看选项），在当前 `. /` 目录以非交互模式进行初始化。
- **原子化与组件化开发**：遵循“原子设计（Atomic Design）”理念，将界面自底向上拆解为原子（基础组件如 Button、Input）、分子（如 SearchBar）与有机体（完整业务模块）。所有界面拼装必须基于预定义的组件，严禁在业务页面编写重复杂乱的样式代码。

## 2. 设计美学与视觉层级（Visual Hierarchy）

要求界面不仅具备可用性，必须让用户在视觉上产生惊艳（WOW）的第一印象，拒绝粗糙的 MVP（最小可行性产品）风格。

- **色彩与模式约束 (Design Tokens)**：
  - 严禁代码中硬编码颜色值（如 `#FFF`, `rgb`），必须抽象为语义化 CSS 变量（如 `--color-primary`, `--bg-surface`）。
  - 弃用通用色系（纯红/纯蓝），采取基于 HSL 的协调色板，深度支持具有质感的暗黑模式（Dark Mode）。
- **排版刻度 (Typography)**：
  - 摒弃浏览器默认字体，引入现代无衬线字体（如 Inter, Roboto, Outfit, San Francisco）。
  - 字号、字重与行高必须遵循离散化的设计系统刻度（例如限制在特定的 Header 与 Body 体系中），并利用不同灰度（而不是单纯加粗放大）来拉开主次文本的层级对比率。
- **空间与留白 (Whitespace & Grid)**：
  - 在布局上依赖于基础网格（通常为 4px 或 8px 的倍数）制定一切 Margin 与 Padding 规范。
  - 优先使用大面积的留白（负空间）和版式对齐来组织信息亲密性，替代生硬死板的边框分割。

## 3. 吸纳行业顶级系统理念

在设计特定组件与交互形态时，默认吸收并遵循以下界面的成熟心理预期：
- **Apple HIG（人机交互指南）**：强调清晰、留白、环境模糊（毛玻璃 Glassmorphism）以及极度平滑真实的过渡微动画。
- **Google Material Design**：遵循物理深度与阴影（Elevation）规范来暗示元素的层级与可点击性。
- **Alibaba Ant Design / Microsoft Fluent**：在中后台与数据密集页面中，强调信息排列的确定性、表单的严格对齐以及利用光感（Light）和材质（Acrylic）跨终端适配。
- **Tailwind UI 理念**：通过强约束的工具类或刻度体系限制多余的设计决策，迫使视觉达成一致。

## 4. 动态交互与微反馈（Micro-interactions）

设计必须感觉充满活力且极具响应性。
- **全状态反馈**：所有交互元素（按钮、链接、卡片）必须提供显式的 `:hover`, `:focus`, `:active` 与 `:disabled` 状态。
- **平滑过渡**：状态切换和页面变动拒绝生硬的跃变。必须包含逻辑统一的微动画过渡机制（如 `< 300ms`, `ease-in-out` 的转场效果）。
- **无占位图**：拒绝在界面中使用空白的 Placeholder。当需要表达意图时，必须通过生成高度拟真的演示图像或骨架屏（Skeleton）。

## 5. 无障碍 (A11y) 与 SEO 工程规范

所有的设计必须建立在代码友好、程序可读的基础之上：
- **可读性**：遵循 WCAG 规范，确保文本色与背景色拥有满足标准的最小对比度（4.5:1）。并强制为键盘操作和屏幕阅读器保留聚焦轮廓（Focus Ring）。
- **SEO 标配**：各页面必须配备包含极佳描述的 `<title>` 和 `meta description` 标签。
- **语义完整**：保持唯一的 `<h1>` 标签进行层级递进；使用正确的 HTML5 语义化标签（如 `<main>`, `<nav>`, `<article>`, `<section>`）。
- **可测试性**：需为所有互动节点绑定具有描述性且唯一的 DOM `ID` 属性以支撑自动化测试工具及浏览器脚本调用。
